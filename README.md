# Better WebClass

WebClass（大学向け LMS）の利便性を向上させる Chrome 拡張機能です。

## 機能

### ファイルハンドラ
- 資料閲覧ページに **↓ DL** / **↓ 科目名DL** ボタンを追加し、正しいファイル名でダウンロード可能に
- 課題・メッセージの PDF をモーダルで表示

### UI 改善
- テーブルの交互着色
- フローティングの「トップへ戻る」ボタン
- コンテンツグループの折りたたみ／展開（アイテム数・期限バッジを表示）

### 課題トラッカー
- 課題・レポート・自習の締め切りまでの残日数をバッジで表示（今日中 / 明日まで / あとN日 / 期限切れ / 提出済）
- 残り時間を時間・分単位で表示
- 種別が「資料」の項目は対象外

### コース一覧への課題情報表示
- 時間割・コース一覧に未提出件数と最短締切をリアルタイム表示
- コースページを訪問するたびに自動でキャッシュを更新

### 自動更新
- バックグラウンドでコースを1つずつ（1秒間隔）巡回し、課題データをキャッシュ
- 更新間隔：1時間 / 3時間 / 6時間 / 9時間 / 12時間から選択
- ポップアップの「今すぐ更新」ボタンで手動更新も可能（自動更新オフでも利用可）
- ポップアップで進捗をリアルタイム確認（N / M コース完了）

## セットアップ

1. ポップアップを開き「WebClass URL」欄に自分の大学の WebClass URL を入力して保存
2. 権限の許可ダイアログが表示されたら「許可」を選択
3. コース一覧ページを開くと課題情報の収集が始まります

> **注意:** 自動更新・手動更新でコース一覧の課題情報を取得するには、事前にコース一覧ページを一度開いてコース URL をキャッシュする必要があります。

## インストール（開発版）

```bash
git clone https://github.com/liquidcatmofu/better-webclass.git
cd better-webclass
npm install
npm run build
```

1. Chrome で `chrome://extensions` を開く
2. 右上の「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. `dist/` フォルダを選択

## 開発

```bash
npm run dev   # ウォッチモード（変更を検知して自動ビルド）
npm run build # プロダクションビルド
```
